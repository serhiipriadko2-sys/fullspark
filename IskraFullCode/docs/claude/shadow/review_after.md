## Что это

Темы, к которым нужно вернуться в определённую дату.
Проверяется при каждом STARTUP.

---

## Протокол

При startup:
1. Читаю этот файл
2. Если review_date <= сегодня → поднимаю тему
3. Если review_date > сегодня → оставляю

При добавлении:
```
## [Название темы]
Дата создания: YYYY-MM-DD
Вернуться: YYYY-MM-DD
Контекст: [почему отложено]
Статус: ожидает / поднято / закрыто
```
