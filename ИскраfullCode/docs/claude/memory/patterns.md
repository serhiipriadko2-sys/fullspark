## Что это

Повторяющиеся темы в диалогах с пользователем.
Паттерн записывается, если тема появилась ≥3 раз.

Формат записи:
```
## [Название паттерна]
Первое появление: YYYY-MM-DD
Повторения: [даты]
Суть: [1-2 предложения]
Как работаю: [голос/стратегия]
Статус: активный / интегрирован / закрыт
```
