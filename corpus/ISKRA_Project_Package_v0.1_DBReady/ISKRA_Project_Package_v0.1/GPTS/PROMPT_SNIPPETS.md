# PROMPT_SNIPPETS — быстрые вставки (Iskra / PI)

## 1) «Сделай ADR (Decision Record)»
**Контекст:** …
**Решение (∆):** …
**Почему:** …
**Альтернативы:** …
**Риски:** …
**D (первый шаг):** …
**Λ (пересмотр):** …

## 2) «Research Note» (PI)
**RQ:** …
**Гипотезы:** H1…
**Операционализация:** метрики/порог/базовая линия
**Дизайн:** …
**Данные:** источник/выборка/срез
**Критерий успеха:** …
**Артефакт:** таблица/скрипт/отчёт

## 3) «SIFT факт‑чек»
- **Stop:** какой тезис проверяем
- **Investigate:** кто источник и каковы интересы
- **Find better coverage:** 2–3 независимых источника
- **Trace:** первоисточник/документ/данные

## 4) «Раздели контуры»
Сделай два раздела:
- **Инженерный контур:** спеки/шаги/критерии
- **Ритуальный контур:** состояние/тень/инвариант

## 5) «Шаблон DoD (Definition of Done)»
- Артефакт создан (ссылка/файл)
- Проверка пройдена (тест/евал)
- Риски отмечены
- След решения (ADR/∆DΩΛ) записан
