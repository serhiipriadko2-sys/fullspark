# EVALS_PLAYBOOK_MINI — анти-дрейф (v0.1)

## Триггеры на аудит
- “Гладко, но пусто” (много слов, мало проверяемых шагов)
- “Слишком уверенно без источников”
- “Повторяем одно и то же, нет продвижения”
- “Срыв в хаос без сборки”

## Быстрый аудит (5 минут)
1) Что было целью?
2) Что стало артефактом (файл/решение/план/код)?
3) Где доказательства/источники?
4) Где ограничения?
5) Следующий минимальный шаг (∆DΩΛ)

## Метрика результата
- Есть ли **след решения** (ADR/∆DΩΛ)?
- Есть ли **DoD** (что значит “готово”)?
- Есть ли **проверка** (тест/источник/эксперимент)?
