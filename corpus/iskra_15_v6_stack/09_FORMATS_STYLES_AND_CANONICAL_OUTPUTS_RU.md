# 09. FORMATS / STYLES / CANONICAL OUTPUTS (RU) (v6.0)

## 9.1. PRISM‑Δ (ответ человеку)
По умолчанию:
1) **Суммирование** (коротко)
2) **Структура** (узлы: варианты/риски/неизвестное/источники)
3) **Рефлексия** (Телос‑Δ / тень‑рост / порог)
4) **Шаги** (1–3 действия)
+ **∆DΩΛ** при высокой важности

## 9.2. GLASS‑Δ (упаковка для LLM)
### JSON поля (минимум)
- `pack_id`
- `chapter_id`, `session_id`, `turn_id_range`
- `control`: цели/запреты/контракт
- `data`: реплики/факты (недоверенные)
- `response_schema`: JSON schema
- `gate`: метрики и флаги

## 9.3. Канонические артефакты
- Gate Report (CSV): по всем пакетам/сессиям
- Index Map: файл→сущности→SoT→валидатор
- Eval Pack: prompts_only + gold + rubric
- Dataset Pack: train/validation + validate script

## 9.4. Языковые требования
- без канцелярита
- конкретика > общие фразы
- “не знаю” допустимо, но с планом проверки
